Projects

Arch Linux is a distrobution of the Linux operating system with a primary focus on simplicity as well as customization. Arch Linux has no easy installation method. As a result, much of the operating system must be configured by the user. After configuring Arch Linux to my likings, I decided to try and optimize the operating system on my machine. I was able to get my custom configuration to use just under 300MB of memory when idle. I also shortened boot time by customizing the kernel for my PC. Right now, I am in the process of automating installation and configuration of certain files/programs in order to have a backup and/or share this with others.
